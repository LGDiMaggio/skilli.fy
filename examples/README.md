# Examples

This directory contains two complete example workflows to demonstrate Skillify's end-to-end pipeline.

## npm-release

A typical npm package release workflow:
- **[recording.jsonl](npm-release/recording.jsonl)** — Raw events captured during a real release (clone → test → bump → build → publish → GitHub release)
- **[generated-skill/](npm-release/generated-skill/)** — The Claude Skill generated by `skillify generate`
  - `SKILL.md` with full instructions, parameters, examples, troubleshooting
  - `references/mcp-github-setup.md` — auto-generated MCP setup guide

**MCP detected:** GitHub (from `git push`, `gh release`, `github.com` hostname)

## notion-report

A metrics reporting workflow with Notion + Slack integration:
- **[recording.jsonl](notion-report/recording.jsonl)** — Events from a report generation workflow
- **[generated-skill/](notion-report/generated-skill/)** — Generated skill with Notion + Slack MCP prerequisites

**MCP detected:** Notion (from `notion.so`, Notion app), Slack (from Slack notification script)

## Try it yourself

```bash
# Generate a skill from the npm-release example recording
cd examples/npm-release
skillify generate <session-id>

# Or manually validate the generated skill
skillify validate examples/npm-release/generated-skill
```
